Installing and Building Quink

To install Quink ready for a build you need the following pieces of software on your system:

  • node.js (https://github.com/joyent/node/wiki/installation)
  • npm (the Node Package manager - https://www.npmjs.org/doc/README.html)
Then follow these steps:
  1. check out the source code into <srcdir>
  2. cd <srcdir>/build
  3. ./install.sh
  4. ./build.sh
    This will use npm to pull down the node packages on which Quink depends and then run the Quink build script leaving the final built version of Quink in <srcdir>/prebuilt.
